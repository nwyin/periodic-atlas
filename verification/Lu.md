## Notes

`elements/Lu.yaml` was created in this verification pass because no prior `elements/Lu.yaml` existed in the worktree or main branch at the time of task execution. The YAML was authored following established patterns from Yb.yaml, Tm.yaml, and Gd.yaml, using project hive notes from worker w-d15a7b335163 which documented key Lu data patterns, then immediately cross-checked against USGS MCS 2025 (PDF extracted via pdftotext).

**Basket share conservatism:** The 0.05% Lu basket share is a deliberately conservative central estimate consistent with hive notes from w-d15a7b335163. Ion adsorption clay deposits in southern China and Myanmar/Thailand yield Lu₂O₃ content of ~0.05–0.15% of total REO, while bastnaesite mines (Mountain Pass, Bayan Obo) yield only ~0.01–0.05%. The global weighted average of 0.05% reflects the mix of Chinese IAC and bastnaesite production; actual Lu output from HREE-rich IAC sources is likely higher.

**Prices:** Confirmed absent from USGS MCS 2025. The prices[] field is intentionally empty per the hive notes pattern for elements not in the USGS salient statistics price table (Ce, Dy, Eu, La, mischmetal, Nd, Tb only). No other source for Lu₂O₃ prices was cited in this file.

**MOFCOM export controls source:** The April 2025 China export controls event uses source_id `mofcom_2025_export_controls` with url=null, as required by hive notes from w-d15a7b335163. Lu is directly and explicitly named in this controlled list (unlike Tm, Er, Ho, Yb which were NOT named). This event occurred after USGS MCS 2025 publication (January 2025) and is not documented in the primary USGS source. Secondary corroboration from multiple hive workers confirms the event.

**Lu directly named vs. not named:** Lu is one of the seven directly named elements in China's April 2025 HREE export controls: Sm, Gd, Tb, Dy, **Lu**, Sc, Y. This distinguishes Lu from Tm, Yb, Er, and Ho, which were NOT directly named. The criticality.notes and geopolitical_events section reflect this distinction clearly.

**No discrepancies found:** All USGS-cited numeric values (country REO totals, world totals, import source percentages, reserve figures, domestic production, net import reliance) are confirmed in the pdftotext-extracted content of the USGS MCS 2025 Rare Earths chapter.
