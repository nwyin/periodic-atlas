## Notes

**Yb.yaml file origin:** Yb.yaml was authored on branch `agent/worker-ee2410510196` (commit 1e772f0) and was not yet merged to `main` at the time of this verification pass. The file was cherry-picked into this worktree for verification purposes.

**Basket share derivation:** All Yb-specific production and reserve tonnages are derived by applying a 0.5% global basket share to USGS-reported aggregate REO totals. This basket share is an industry estimate, not stated by USGS. For HREE-enriched ion adsorption clay deposits (southern China, Myanmar, Thailand), actual Yb₂O₃ content is ~1–2% of REO, substantially above the global average. The 0.5% central estimate likely understates Yb output from HREE-dominant IAC sources and overstates it from LREE-dominant mines (Mountain Pass, Mt. Weld).

**Laser end-use claim discrepancy (×2):** end_uses[yb_fiber_lasers_industrial] and end_uses[yb_yag_ultrashort_pulse_lasers] both contain the statement "USGS MCS 2025 lists lasers as a major REE end use." This is FALSE. The USGS MCS 2025 Domestic Production and Use section (p.144) lists: catalysts (leading), permanent magnets (embedded in finished goods imports), ceramics and glass, metallurgical applications and alloys, and polishing. Lasers are not named. This same error was flagged for Ho.yaml by worker w-a3a44095dd36. The share_pct values themselves remain inferred (not discrepancies), since USGS does not publish per-element end-use percentages; only the falsely cited attribution is a discrepancy.

**EU criticality source misattribution (×2):** `criticality.eu_crm_source_id` and `criticality.eu_strategic_source_id` are both set to `usgs_mcs_2025_rare_earths` in Yb.yaml, which is incorrect. The EU Critical Raw Materials and Strategic Raw Materials designations for rare earth elements (including heavy REEs like Yb) are established by EU Regulation 2024/1252 (Critical Raw Materials Act). Analogous element files (Tm.yaml, Ho.yaml) correctly use `eu_crma_2024` as the source for these fields. USGS MCS 2025 does not cite or document the EU CRMA designation.

**Prices:** Confirmed absent from USGS MCS 2025 Salient Statistics table. The prices: [] field is correctly empty; the table covers only Ce, Dy, Eu, La, mischmetal, Nd, and Tb.

**MOFCOM export controls source:** The April 2025 China export controls event uses source_id `china_mofcom_export_controls_2025` with url=null. This event occurred after USGS MCS 2025 publication (January 2025) and is not documented in the primary USGS source. The event is consistent with hive-note intel confirming Yb was NOT directly named in the April 2025 controls (which targeted Sm, Gd, Tb, Dy, Lu, Sc, Y).

**Vietnam production:** USGS table shows Vietnam 2024 = 300 t (unchanged from 2023 at 300 t). The Yb.yaml notes state "Vietnam produces only ~300 t REO per year" — this is confirmed.
